## 🌐 Live Demo
You can test the live simulation here: **[https://hkzabi139.github.io/Task-4/]**


Key Features
Real-time Interaction Tracking: Monitors mouse speed, click frequency, and typing patterns to detect stress or calm.
Dynamic Stability Score: A live score (0-100) that adjusts based on interaction intensity.
Glassmorphic UI: A modern, responsive dashboard with a dynamic mood-based glow effect.
Live Analytics Graph: Integrated Chart.js to visualize the stability score history in real-time.
Interactive Event Log: Displays the last 10 tracked events with timestamps and score impacts.
Session Export: Option to download the entire activity log as a JSON file for further analysis.

🛠️ Technical Stack
Frontend: HTML5, CSS3 (Custom Glassmorphism & Animations)
Logic: Vanilla JavaScript (Event listeners & State management)
Visualization: Chart.js Library

🎯 Mood Categories
0–20: Overstimulated (Red Glow)
21–40: Frustrated (Orange Glow)
41–60: Neutral (Yellow Glow)
61–80: Calm (Blue Glow)
81–100: Deep Focus (Green Glow)
